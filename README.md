> *Feel the taste of the most popular Japanese foods from anywhere and anytime.*

🌐 **Live Demo:** [abdulqahardawood-oss.github.io/Sushiweb](https://abdulqahardawood-oss.github.io/Sushiweb/)

## 📖 Overview

**Sushiman** is a modern, elegantly designed Japanese food delivery website built with HTML, CSS, and JavaScript. It features a clean split-layout hero section, illustrated food items, customer testimonials, and a smooth browsing experience — all wrapped in a refined dark-rose and off-white aesthetic inspired by Japanese culinary culture.

---

## ✨ Features

- **Hero Section** — Bold split layout with Japanese kanji typography and a prominent "Order Now" CTA
- **Social Proof** — 24k+ Happy Customers stat with inline customer testimonial card
- **About Us Section** — Mission statement with bilingual labels (`About Us / 私たちに関しては`)
- **Food Menu Cards** — Illustrated dish cards (Chezu Sushi, Originale Sushi, Ramen Legendo) with ratings and pricing
- **Navigation Bar** — Clean top nav with links: Menu, Food, Services, About Us + search icon
- **Responsive Design** — Optimized for desktop and tablet viewports

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- A local development server (e.g. [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for VS Code)

### Running Locally

```bash
# Clone the repository
git clone https://github.com/your-username/sushiman.git

# Navigate into the project directory
cd sushiman

# Open with Live Server or simply open index.html in your browser
open index.html
```

The site runs at `http://127.0.0.1:5501/index.html` when using Live Server.
